# Netlify setup

Add these GitHub repository secrets before running the frontend deployment workflow:

- NETLIFY_AUTH_TOKEN
- NETLIFY_SITE_ID

Netlify setup steps:

1. Create or select a site in Netlify.
2. Open Site configuration → General → Site details to find the Site ID.
3. Generate a personal access token in Netlify user settings.
4. Add both values to GitHub repository secrets.
